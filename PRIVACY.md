# Privacy Policy — Flight Grid Extender

_Last updated: 2026-06-21_

Flight Grid Extender does **not** collect, store, transmit, sell, or share any
personal or user data.

- The extension runs only on Google Flights pages under
  `https://www.google.com/travel/flights`.
- All network requests it makes go to the same `www.google.com` origin the user is
  already browsing, using the browser's existing session — exactly the requests the
  page itself would make to load price data. No data is sent to the developer or any
  third party.
- No analytics, no tracking, no telemetry, no external/remote code.
- No data is persisted; results live only in the open page and disappear when it is
  closed.

Because no data is collected, there is nothing to access, export, or delete.
